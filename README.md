# Store React App

Веб-додаток на React, що імітує роботу невеликого інтернет-магазину. Користувач може переглядати список товарів, використовувати фільтрацію, сортування та пагінацію. Адміністративна частина дозволяє додавати, редагувати та видаляти товари через API.

## Інструкція зі встановлення та запуску

### 1. Клонування репозиторію

git clone https://github.com/admsldy/store-react-app.git

### 2. Перехід у папку проєкту
cd vite-project

### 3. Встановлення залежностей
npm install

### 4. Запуск проєкту
npm run dev

Після запуску додаток буде доступний за адресою:

http://localhost:5173
