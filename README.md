# Store React App

Веб-додаток на React, що імітує роботу невеликого інтернет-магазину. Користувач може переглядати список товарів, використовувати фільтрацію, сортування та пагінацію. Адміністративна частина дозволяє додавати, редагувати та видаляти товари через API.


## Функціональність

### Загальні можливості
- Отримання даних з API
- Робота з асинхронними запитами
- Відображення станів завантаження (loading)
- Обробка помилок при роботі з API
- Повідомлення про відсутність даних

### Користувацька частина (Products)
- Перегляд списку товарів
- Пошук товарів за назвою (мінімум 3 символи)
- Сортування товарів
- Пагінація з номерами сторінок
- Повідомлення у разі відсутності результату пошуку

### Адміністративна частина (Admin Products)
- Містить весь функціонал користувацької частини (пошук товарів, сортування, пагінація)
- Додавання нових товарів
- Редагування товарів
- Видалення товарів

## Інструкція зі встановлення та запуску

### 1. Клонування репозиторію

git clone https://github.com/admsldy/store-react-app.git

### 2. Перехід у папку проєкту
cd vite-project

### 3. Встановлення залежностей
npm install

### 4. Запуск проєкту
npm run dev

Після запуску додаток буде доступний за адресою:

http://localhost:5173
